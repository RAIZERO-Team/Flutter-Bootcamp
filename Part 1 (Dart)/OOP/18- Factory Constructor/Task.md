## CONSTRUCTOR

### 07- Factory Constructor

```
CREATE AN INTERFACE CALLED BOTTLE AND ADD A METHOD 
TO IT CALLED OPEN().

CREATE A CLASS CALLED COKEBOTTLE AND IMPLEMENT 
THE BOTTLE AND PRINT THE MESSAGE “COKE BOTTLE IS OPENED”.

ADD A FACTORY CONSTRUCTOR TO BOTTLE AND RETURN THE OBJECT 
OF COKEBOTTLE.

INSTANTIATE COKEBOTTLE USING THE FACTORY 
CONSTRUCTOR AND CALL THE OPEN() ON THE OBJECT.
```
