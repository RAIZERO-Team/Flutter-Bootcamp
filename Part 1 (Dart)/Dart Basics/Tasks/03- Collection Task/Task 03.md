## Task 4: Collection Task

Create a variable `students` and it should
to contain the following variables:

```
students => {
  Student1,
  Student2 => {
    name => 'STUDENT_NAME',
    age => STUDENT_AGE,
    subjects => {
      SUBJECT1,
      SUBJECT2 => {
        title => SUBJECT_TITLE,
        lesson_count => LESSON_COUNT
      }
    }
  }
}
```
